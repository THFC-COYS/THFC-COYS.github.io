Featured Project: Day in the Metaverse

This project explores how spatial computing can transform remote education. The video demonstrates using Meta Workrooms as a dynamic virtual classroom for more engaging and collaborative learning experiences.

Tech Stack: Meta Workrooms, Meta Quest 3, OBS for recording.

Hypothesis: Immersive, shared virtual spaces can significantly increase feelings of presence and reduce isolation for remote learners compared to traditional video conferencing.

Method: Conducted A/B testing with student groups; one cohort used traditional Zoom for breakout sessions, the other used Meta Workrooms. Collected qualitative feedback via surveys and quantitative data on session length and re-engagement.

Insight: The Workrooms cohort reported a 40% higher sense of "being there" and collaborated for 25% longer on average, validating the hypothesis that immersion drives engagement.
